### Install:
- kali & ubuntu

Install tool packags
```bash
$ sudo apt install python3
```
```shell
$ sudo apt install git
```
```bash
$ sudo apt install python3-pip
```
Download tool
```bash
$ git clone https://github.com/limearch/HackerMode.git
```

Install
```bash
$ sudo python3 -B HackerMode/ install
```
- termux
```shell
# Install tool packags
$ pkg install python
$ pkg install git

# Download tool
$ git clone https://github.com/Arab-developers/HackerMode

# Install
$ python3 -B HackerMode/ install
```

### Docs:
How to use HackerMode?
```shell
# after install
# write in terminal HackerMode
# to run the tool
$ HackerMode

# to check HackerMode package
$ HackerMode check

# to update HackerMode and packages
$ HackerMode update
$ HackerMode upgread
```
